# 🌹 The Rosa — Restaurante

Site institucional fictício para um restaurante de alta categoria, desenvolvido como projeto de estudo em HTML e CSS.

## Sobre o projeto

The Rosa é um projeto de interface web com foco em identidade visual sofisticada. O desafio principal foi traduzir a atmosfera de um restaurante premium em uma página estática trabalhando tipografia, paleta de cores, hierarquia visual e uso de animalçoes simples
## O que foi praticado

- Identidade visual aplicada ao front-end
- Tipografia e hierarquia visual com CSS
- Estilização de layouts elegantes com animações em JavaScript
- Estruturação semântica de páginas institucionais
